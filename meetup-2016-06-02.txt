Quelques sujets possibles:


Benoît:
- Utilisation de WP-CLI (installation, rechercher / remplacer dans la DB, ...)
- Création d'un plugin simple pour ajouter un Custom Post Types et des Custom Taxonomies
- le "choix" d'une license. 

Luca: 
I can speak (in French) about GIT and WordPress, it's not a relay dev technique but more how to use a dev technique to install and update WordPress (I applied and extended the technique described in this post https://davidwinter.me/install-and-manage-wordpres    s-with-git/).

Manuel:
"Let shortcodes do the work for you!" 
Exemple 1: usage du shortcode [gallery] pour (a) ne pas devoir se fatiguer avec le markup et (b) rendre le code plus pérenne. Exemple 2: embed de vidéo youtube ou vimeo... on fournit simplement l'URL et on y applique le filtre qui produit l'embed.

Manuel:
GRUNT
Utilisation de GRUNT pour faire la compilation & concaténation du CSS / JS. Problème: il faut renommer à chaque fois la fonction qui "enqueue" le CSS. Astuce: GRUNT peut le faire! :)

Manuel:
Override des langues (thème et plugins)
Utilisation du code proposé par Ulrich Pogson.
Utilisation de l'application POEdit.
